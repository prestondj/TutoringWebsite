Accessibility Statement

Last updated: June 2026

PDJ Tutoring is committed to ensuring that this website is accessible to as many people as possible, regardless of ability, technology, or access needs. We aim to meet the requirements of the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA.

Our Commitment

We are actively working to make this website more accessible and usable for all users, including those who:

  • Use screen readers or other assistive technologies
  • Navigate using only a keyboard
  • Have visual impairments or colour sensitivity
  • Require larger text or simplified layouts

Accessibility Features on This Website

This website currently includes the following accessibility features:

  • Keyboard navigation support with visible focus indicators
  • Skip-to-content link to bypass repeated navigation
  • Adjustable accessibility mode with increased text size and spacing
  • Reduced motion support to minimise animations for users who prefer it
  • Consistent semantic structure using headings, sections, and landmarks

Ongoing Improvements

Accessibility is an ongoing priority for this website. We regularly review design and functionality to improve usability and ensure compliance with best practices.

Feedback

If you experience any accessibility issues or require information in a different format, please get in touch using the contact form on this website. We will do our best to respond promptly and provide alternative access where possible.

Compliance Status

This website is partially compliant with WCAG 2.1 Level AA due to the limitations listed above. We are actively working toward full compliance through ongoing improvements.
